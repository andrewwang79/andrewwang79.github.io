# 权威
任何一个领导都必须对部下具有权威。

什么是权威？权威就是领导者以权力强制的社会合法性为基础将自己的意志强加给下属，下属对权力实体——领导者的一种服从。因此，权威的特征是强制与服从。

权威的确立有三种方式：第一种是靠权力压迫，即力服；第二种是靠才干与智慧，即才服；第三种是靠人格的魅力，即德服。三种方式相辅相成，在不同的情况下，各有优势，但是，总的来说，应该是德服为上、才服为中、力服为下。
