# 区别
- 数据分析：先决而后知。

常规数据分析侧重于解决数据挖掘以外的问题：如描述性统计、交叉报表、假设检验等。

- 数据挖掘：先知而后决。有模型就是挖掘。

数据挖掘主要侧重解决四类问题：分类、聚类、关联、预测
